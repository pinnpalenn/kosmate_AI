## ✨ Fitur Utama
- 📖 **Dokumentasi Terstruktur:** Memuat dokumen SKPL lengkap (tabel karakteristik, istilah, use case).
- 🌐 **Web-Based Viewer:** File HTML agar dokumen mudah diakses dan dibaca langsung melalui browser.
- 📊 **Visualisasi Diagram:** Terdapat rancangan diagram yang disematkan secara interaktif.
- 📄 **Format PDF:** Menyediakan versi *hardcopy/printable* (SKPL_KostMate_Alvin Afalen.pdf).

Berikut adalah struktur folder dan organisasi file dari repository dokumentasi ini:
```text

├── styles/                         # File CSS untuk memperindah tampilan web SKPL
